習慣四 · Habit 4
雙贏思維
合作不是妥協。雙贏是找到一條讓雙方都真正更好的路——不是你讓我，我讓你，而是一起找到第三條路。
核心概念｜情感帳戶

柯維說，每一段關係都有一個情感帳戶。存款：讓對方感覺被尊重、被理解、被信任。提款：失信、輕視、只顧自己。

帳戶餘額高的時候，就算你說錯一句話，對方也不會太在意。帳戶透支的時候，你說什麼都是錯的。

雙贏不是談判技巧，它建立在情感帳戶存款夠多的基礎上。你的帳戶現在是正數還是負數？

「雙贏不是一種技巧，而是一種對人際互動的全面哲學。」

— 史蒂芬．柯維《與成功有約》
你現在在哪裡？
想想你最重要的一段工作關係或合作關係。
✓
談合作或協商時，我會先想「怎麼讓雙方都更好」 不是只想著我能拿到什麼。
✓
我說到做到，對方可以信任我的承諾 我在重要關係的情感帳戶裡，存款多於提款。
✓
遇到利益衝突，我會找第三條路而不是堅持我的版本 我願意接受最後的結果可能跟我一開始想的不一樣。
✓
我不會為了贏得眼前的爭論，而傷害長期的關係 我知道關係比單次結果更重要。
✓
如果找不到雙贏的方案，我願意選擇「無交易」 強迫對方接受對他不好的結果，不是我的風格。

習慣五 · Habit 5
知彼解己
先理解別人，再讓別人理解你。當一個人感覺被真正理解了，他才有空間聽你說話。
核心概念｜移情聆聽

柯維說，大多數人聽話的方式叫做「自傳式回應」——對方還沒說完，你就已經在腦袋裡準備你的故事、你的建議、你的判斷了。

移情聆聽不是要你同意對方，而是讓對方感覺被真正理解。這需要你暫時放下自己的思維定式，真的站在對方的角度看事情。

你上一次真正聽完別人說話，是什麼時候？

「先尋求理解，再尋求被理解。」

— 史蒂芬．柯維《與成功有約》
你現在在哪裡？
想想你最近的對話和溝通情況。
✓
聽別人說話時，我不會一邊聽一邊準備我的回應 我真的讓對方說完，才開始思考我要說什麼。
✓
我可以用對方的角度重新說一遍他的立場 不是嘲諷，是真的理解他為什麼這樣想。
✓
跟我想法很不一樣的人說話，我不會急著反駁 我會先問：「你為什麼這樣想？」
✓
身邊的人覺得跟我說話很輕鬆，願意分享真實想法 他們不用擔心被評判。
✓
我知道自己什麼時候在用「自傳式回應」 我有能力在那個當下停下來，重新聽。
知彼解己程度 0 / 5

習慣六 · Habit 6
統合綜效
整體大於部分之和。差異不是你要解決的問題，差異是你可以使用的材料。真正的創造，發生在有分歧的地方。
核心概念

統合綜效需要三個條件：

第一：你要夠有安全感——你不需要對方同意你，才能把自己的想法說出來。
第二：你要真的對對方的觀點感到好奇，不是為了找漏洞，而是真的想搞懂。
第三：你要願意接受，最後的結果可能跟你一開始想的完全不一樣——而那個結果，往往比你原本的想法還要好。

你上一次因為有人跟你想法不同，而創造出更好的結果，是什麼時候？

「如果兩個人意見完全相同，其中一個人是多餘的。」

— 史蒂芬．柯維《與成功有約》
你現在在哪裡？
想想你跟別人合作或討論的狀態。
✓
遇到跟我想法不同的人，我的第一反應是好奇而不是防禦 我想知道他為什麼這樣想，不是急著說他哪裡錯了。
✓
我不需要對方同意我，也能把我的想法說清楚 我的安全感來自內在，不是來自別人的認可。
✓
我有過因為不同意見，最後找到比原本更好方案的經驗 我知道差異可以創造價值。
✓
我會主動找跟我背景、觀點不同的人討論重要決策 我不只待在同溫層裡尋求確認。
✓
合作之後的結果，通常比我一個人想到的還要好 我真的相信一加一可以大於二。
統合綜效程度 0 / 5

習慣七 · Habit 7
不斷更新
磨刀不誤砍柴工。習慣七是所有習慣的基礎——你需要持續更新自己，前六個習慣才不會退化。
核心概念｜四個維度

柯維說，不斷更新需要投資四個面向：

生理：運動、飲食、休息——身體是一切的基礎
心智：閱讀、學習、思考——讓大腦保持銳利
社會情感：關係、服務、移情聆聽——情感帳戶的充電
靈性：價值觀、使命、靜思——確認你走在對的路上

四個維度都需要主動投資，不是等到撐不住才休息。

「磨利你的鋸子，是你能做的最重要的事——它能讓其他所有事情發生。」

— 史蒂芬．柯維《與成功有約》
你現在在哪裡？
想想你最近一個月，有沒有主動投資這四個面向。
✓
我有規律的運動習慣，不是心情好才去 我把身體當成長期資產在經營。
✓
我有固定的閱讀或學習時間，不是有空才讀 我主動保護這段時間，不讓其他事情佔走。
✓
我會定期回顧自己的狀態，不是一路衝到倒下 我知道什麼時候需要充電，然後真的去充電。
✓
我有定期檢視自己的目標和方向的習慣 確認我還在走向我想去的地方。
✓
忙完一個大任務之後，我會給自己時間盤點和復原 不是馬上衝進下一件事，而是先整理這段經歷。
