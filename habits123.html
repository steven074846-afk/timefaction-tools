習慣一 · Habit 1
積極主動
在刺激和回應之間，有一個空間。在那個空間裡，你有選擇的自由。積極主動的人選擇自己的回應——而不是讓環境替他做選擇。
核心概念

柯維說，你可以看一個人怎麼說話，就知道他是積極主動還是消極被動。

消極被動的語言：「我沒辦法」、「都是因為⋯⋯」、「如果不是⋯⋯」
積極主動的語言：「我可以選擇」、「我來想辦法」、「我決定」

還有一個關鍵概念：影響圈 vs. 關注圈。積極主動的人把精力放在自己能控制的事（影響圈），而不是花時間煩惱控制不了的事（關注圈）。

「你沒辦法控制風的方向，但你可以調整帆的角度。」

— 史蒂芬．柯維《與成功有約》
你現在在哪裡？
勾選最近一個月內你有的狀態。誠實面對自己，這裡沒有對錯。
✓
遇到問題，我第一個反應是找解法，不是找藉口 就算外部條件很差，我也會想「在這個限制內，我還能做什麼？」
✓
我說話很少用「因為他⋯⋯所以我沒辦法⋯⋯」 我知道別人的行為和外部環境不能決定我的回應。
✓
我把精力花在我能控制的事情上 不會花大量時間抱怨景氣、公司制度、或別人的態度。
✓
就算結果不如預期，我會先反思自己能改變什麼 不會只歸咎外部原因然後放棄。
✓
我對自己的承諾說到做到 自己答應自己的事，我會認真對待。

習慣二 · Habit 2
以終為始
所有事情都會被創造兩次——第一次在你的腦袋裡，第二次才在現實裡。如果你沒有先想清楚終點，你就只是在跟著別人的劇本走。
核心概念

柯維說，以終為始不是叫你設定模糊的人生夢想，而是讓你在每一個當下都知道：我現在做的這件事，是為了什麼。

每一段經歷、每一份工作、每一個決定——在進去之前，你有沒有先想清楚你要帶走什麼？

沒有終點的努力，只是在消耗自己。

「管理是把梯子架好爬上去，領導則是先確認梯子是否靠在對的牆上。」

— 史蒂芬．柯維《與成功有約》

✓

習慣三 · Habit 3
要事第一
習慣一讓你知道可以選擇，習慣二讓你知道要去哪裡，習慣三讓你真的把時間和精力放在對的地方。
核心概念｜時間管理矩陣

柯維把所有事情分成四格：

第一格｜重要＋緊急：危機、截止日、緊急事件→必須處理，但不能讓它佔滿你的生活
第二格｜重要＋不緊急：學習、健康、關係、規劃→高效能人士最應該投資的格子
第三格｜不重要＋緊急：別人的急事、冗長的行政流程→時間的最大殺手
第四格｜不重要＋不緊急：刷手機、無意義的消遣→直接刪掉

大多數人的問題是：被第三格吃掉太多時間，卻沒有空間投資第二格。

「重點不是把時間表排滿，而是把真正重要的事排進去。」

— 史蒂芬．柯維《與成功有約》
你現在在哪裡？
想一想上個禮拜，你的時間大概花在哪裡。
✓
上週我有花時間在學習或個人成長上（第二格） 不是偶爾，是刻意安排的時間。
✓
我每天知道今天最重要的一件事是什麼 不是什麼都想做，而是有清楚的優先順序。
✓
我可以對不重要的事說「不」 別人的急事不一定是我的急事，我知道怎麼拒絕。
✓
我一天結束的時候，不會覺得忙了一整天但什麼都沒做到 我忙完之後，知道自己往前走了。
✓
我有保護自己休息和恢復精力的時間 不是被逼才休息，是主動安排。
